# Usage
## Requirements

* ``npm``
* ``node``

## Install npm dependencies

```
$ cd cis/scripts/js/
$ npm install
```

### Run it directly using nodejs
```
$ cd cis/scripts/js/
$ node fake-cis-data.js
```

### Install and run script from system path (requires root)

```
$ cd cis/scripts/js/
$ sudo npm link
$ cis-data-faker
```
